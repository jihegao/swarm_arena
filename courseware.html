3. Tick pipeline / Tick 流水线

1. PerceiveWorld builds local, bounded perception. 世界构造局部感知。
2. DecideCreature returns one action. 生物返回单帧动作。
3. SanitizeInvalid values and exceptions are contained. 非法值和异常被隔离。
4. ResolveMovement, energy, eating, reproduction, scoring. 解析移动、能量、进食、繁殖、计分。

The decision API does not expose full world state. 决策 API 不暴露完整世界状态。

4. Creature contract / 生物实现契约

Required shape

  • Subclass Creature or EvolvableCreature.
  • Constructor signature: __init__(self, x, y).
  • Implement decide(self, perception).
  • Import without prompts or side effects.

最低要求

  • 继承 Creature 或 EvolvableCreature。
  • 构造函数接受 x, y。
  • 实现 decide(perception)。
  • 导入时不弹交互、不执行实验。
class RuleSurvivor(Creature):
    def __init__(self, x: float, y: float):
        super().__init__(x, y, "RuleSurvivor", (255, 128, 0))

    def decide(self, perception: Perception) -> Action:
        return Action(perception.self_angle, self.max_speed * 0.5)

5. API details / API 细节

Perception

self_x, self_y, self_angle, self_size, self_energy, self_max_energy, world_width, world_height, nearby_food, nearby_creatures.

Action

target_angle and target_speed drive movement. attack_target_id marks intent only. reproduce=True requests reproduction.

进食规则

大吃小基于碰撞和冷却。只要碰到更小生物并满足冷却，引擎会自动结算。

属性预算

max_speed, max_energy, and vision_radius share a normalized budget capped at 1.5.

6. Training boundaries / 训练边界

GA parameter search / GA 参数搜索

Use when behavior is readable and numbers need tuning. Target creatures should inherit EvolvableCreature and define gene ranges.

.venv/bin/python train.py \
  --method ga \
  --creature creatures/name.py \
  --generations 20 \
  --history-output ga_history.json \
  --output best_params.json

Q-learning exploration / Q-learning 探索

Use for high-level action tradeoffs. Current RL trains LearningCreature; it does not rewrite arbitrary creature files.

.venv/bin/python train.py \
  --method rl \
  --episodes 200 \
  --output best_policy.json
