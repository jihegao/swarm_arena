🎯 游戏目标

⏱ 时间限制
游戏在 5000 tick 后结束（或只剩一种生物时提前结束）。
🏆 胜利条件
存活数量最多的物种中，能量最高的个体获胜。
🦁 核心玩法
抢食物 → 长大 → 吃掉比自己小的对手 → 繁殖后代 → 数量碾压。
📊 计分规则
吃掉对手 +对手max_energy | 吃食物 +amount | 繁殖 +100 | 最终获胜 +500

关键思路：这是一个滚雪球游戏。前期抢到食物 → 体型变大 → 能吃更多对手 → 获得更多能量 → 繁殖更多后代。起步很关键！

📝 你的任务

只需要做两件事：创建类 和 实现 decide()。

最小模板

from __future__ import annotations
import math, os, sys
sys.path.insert(0, os.path.dirname(os.path.dirname(os.path.abspath(__file__))))
from creature import Creature, Perception, Action

class MyCreature(Creature):
    def __init__(self, x: float, y: float):
        super().__init__(x, y, "MyCreature", (255, 128, 0))

    def decide(self, perception: Perception) -> Action:
        return Action(perception.self_angle, self.max_speed)

这段代码创建了一个只会直线前进的橙色生物。你可以在此基础上逐步改进！

👁 Perception — 你能看到什么

每一帧，游戏引擎调用 decide(perception)，传入感知信息：

字段类型说明
self_x / self_yfloat你的坐标
self_anglefloat你的朝向（弧度，0=右，pi/2=下）
self_sizefloat你的体型大小（随能量动态变化）
self_energyfloat你的能量值
self_max_energyfloat你的初始能量值
world_width / heightint世界尺寸
nearby_foodlist[FoodSpot]视野范围内的食物列表
nearby_creatureslist[CreatureSpot]视野范围内的其他生物
FoodSpot
x, y 坐标 | amount 能量值 | size 大小
CreatureSpot
x, y 坐标 | creature_type 类型名 | size 体型 | energy_pct 能量百分比

你只能看到 vision_radius 范围内的东西，范围外的世界不可见！

🎯 Action — 你要做什么

每帧返回一个 Action 对象：

Action(
    target_angle,       # float: 这一帧要朝向的角度（弧度）
    target_speed,       # float: 这一帧的移动速度（不超过 max_speed）
    attack_target_id,   # int | None: 要追击的生物 ID（仅标记意图）
    reproduce,          # bool: 是否繁殖（默认 False）
)
🏃 移动
每帧必须指定 target_angle 和 target_speed。设 0 表示停止。
⚔️ 攻击
碰到比自己小的生物会自动吃掉，无需主动声明攻击！
👶 繁殖
设 reproduce=True 且满足条件时，附近生成同类型子代。

战斗规则很简单：大吃小。没有攻击力和防御力。你只需要移动到比你小的生物旁边就行！

⚙️ 属性系统与点数分配

可配置属性

属性默认最小最大说明
max_speed1.00.54.0最大移动速度
max_energy10030200初始能量值
vision_radius2050300视野半径

点数预算 = 1.5

# 归一化值 = (值 - 最小) / (最大 - 最小)
# 总点数 = 各属性归一化值之和
# 超过 1.5 则等比例缩小

# 示例：Hunter 的分配
speed  = (2.5 - 0.5) / (4.0 - 0.5) = 0.57
energy = (80  - 30) / (200 - 30) = 0.29
vision = (200 - 50) / (300 - 50) = 0.60
总计   = 1.46

速度最关键：能吃到食物就能长大，长大就能吃别人，是滚雪球的核心！

🍖 大吃小规则

战斗规则

  • 碰到比自己体型小的生物，直接吃掉
  • 获得对方全部剩余能量
  • 碰撞范围：距离 < (eater.size + target.size) × 0.6
  • 吃掉后 10 帧冷却

体型公式

size = 6 × sqrt(energy / 100)

👶 繁殖机制

繁殖条件

条件值
能量阈值≥ max_energy × 2.0
繁殖消耗max_energy × 0.5
冷却时间120 帧
子代位置母体周围随机 20px
世界上限600 生物

流程

  1. reproduce=True
  2. 检查能量 ≥ max_energy × 2
  3. 检查冷却为 0
  4. 扣除能量，生成子代
  5. 进入 120 帧冷却，得 100 分

繁殖策略

⏰ 早生策略
能量刚到阈值就生，子代小但数量多。
💪 晚生策略
攒更多能量再生，子代更大更安全。
🔭 时机选择
周围没有天敌时繁殖更安全。

判断是否可繁殖：perception.self_energy >= perception.self_max_energy * 2.0

⚡ 能量系统

能量是生存的核心。能量降到 0 你就死了。

能量消耗（每帧）

来源公式
基础衰减0.1 × (体型 / 6)
移动消耗额外 0.08 × (速度/max_speed) × (体型/6)

体型越大，能量消耗越高！没有食物来源时，适当"减肥"可能比维持巨大体型更持久。

能量恢复

来源效果
🍞 吃食物+ 食物的 amount（8~100）
🦁 吃掉对手+ 对手全部剩余能量
👶 繁殖- max_energy × 0.5

吃食物条件

你与食物的距离 < 你的体型 + 3

👥 内置对手分析

🔴 Hunter（猎人）
速度 2.5 | 能量 80 | 视野 200
追猎视野内最弱的生物，低能量时找食物。能量充足时繁殖。
弱点：初始能量低，前期容易被吃。
🟢 Grazer（食草者）
速度 1.5 | 能量 100 | 视野 150
以吃食物为主，敌人接近 35px 时逃跑。能量超 2.5 倍时繁殖。
弱点：速度最慢，容易被追上。
🔵 PackHunter（群猎者）
速度 2.2 | 能量 80 | 视野 150
寻找同伴，聚在一起时围猎。附近同类 <5 且能量充足时繁殖。落单时逃跑。
弱点：初始能量低，容易被各个击破。
🟡 Scavenger（拾荒者）
速度 3.5 | 能量 60 | 视野 160
全场最快，抢到食物就跑。能量超 3 倍时繁殖。从不主动攻击。
弱点：初始能量最低，谁都能吃它。

💻 完整示例：Survivor

class Survivor(Creature):
    def __init__(self, x: float, y: float):
        super().__init__(x, y, "Survivor", (255, 128, 0))
        self.max_speed = 2.2
        self.max_energy = 100.0
        self.vision_radius = 160.0

    def decide(self, p: Perception) -> Action:
        # 1. 能量充足时繁殖
        if p.self_energy >= p.self_max_energy * 2.5:
            return Action(p.self_angle, 0, reproduce=True)
        # 2. 追杀比自己小的对手
        smaller = [c for c in p.nearby_creatures if c.size < p.self_size]
        if smaller:
            t = min(smaller, key=lambda c: math.hypot(c.x-p.self_x, c.y-p.self_y))
            return Action(math.atan2(t.y-p.self_y, t.x-p.self_x), self.max_speed)
        # 3. 远离比自己大的对手
        bigger = [c for c in p.nearby_creatures if c.size > p.self_size]
        if bigger:
            b = min(bigger, key=lambda c: math.hypot(c.x-p.self_x, c.y-p.self_y))
            if math.hypot(b.x-p.self_x, b.y-p.self_y) < 60:
                flee = math.atan2(p.self_y-b.y, p.self_x-b.x)
                return Action(flee, self.max_speed)
        # 4. 找食物或随机漫步
        if p.nearby_food:
            f = min(p.nearby_food, key=lambda f: math.hypot(f.x-p.self_x, f.y-p.self_y))
            return Action(math.atan2(f.y-p.self_y, f.x-p.self_x), self.max_speed*0.7)
        return Action(p.self_angle+math.sin(p.self_x*0.015)*0.2, self.max_speed*0.4)
